Dalam pembuatan aplikasi Sistem Kasir Toko Novi Mart ini saya mencari banyak referensi dari berbagai media social, setelah saya mendapatkan idenya saya membuat codingan dengan melihat beberapa website seperti boostrap, dan css untuk mempunyai syle.css yang dimana digunakan untuk membuat tampilan lebih menarik.
Sistem kasir pada Toko Novi Mart ini saya buat karena pada zaman yang sekarang ini sudah banyak yang menggunakan computer untuk menginputkan data nya, maka dari itu saya mempunyai ide untuk membuat aplikasi sistem kasir

lalu saya akan menjelaskan progam aplikasi berbasis web php dan mysql 
sebelum itu saya jelaskan pada dashboard utama inputkan belajaan terlebih dahulu baru isi data pelanggan agar tercatat di data laporan, tidak lupa untuk mengisi pembayaran lalu pilih cetak dan disimpan

1.config.phpp digunakan untuk menyambukaan file php dengan database mysql

2.login.php dalam pengembangan sistem kasir digunakan untuk membuat halaman login yang memungkinkan pengguna   untuk melakukan autentikasi dan mengakses fitur-fitur yang terkait dengan sistem kasir.

3.footer.php dalam sistem kasir digunakan untuk mengatur tampilan dan konten footer secara konsisten di semua halaman situs kasir.

4.header.php dalam konteks pengembangan sistem kasir berbasis web digunakan untuk mengatur tampilan bagian header atau kepala pada setiap halaman situs kasir.

5.index.php dalam pengembangan sistem kasir biasanya digunakan sebagai halaman utama atau halaman utama dari situs kasir
6.kategori.php digunakan untuk mendata produk jenis apa saja yang akan dijual misalnya seperti alat tulis, makanan,skincare dan lainnya

7.produk.php dimana digunakan untuk data produk seperti pada kategori makanan ada “oreo” dan kategori obat ada “paracetamol” , sebelum memasukan data produk isilah kategori yang sesuai terlebih dahulu

8.laporan.php digunakan untuk mendata pelanggan berbelanja apa saja dan menghitung data keuntungan yang dimana diisikan pada saat berbelanja, lalu inputkan belanjaan

9.pengaturan.php ini mengatur siapa yang mengoperasikan aplikasi kasir tersebut lalu digunakan untuk login 

10.pelanggan.php digunakan untuk mendata pelanggan agar bisa mengimputka kedalam laporan.php agar nama tampil di laporan kasir

11.detail.php digunakan untuk dapat digunakan untuk menampilkan informasi lengkap tentang produk tertentu, seperti nama produk, deskripsi, stok, dan atribut lainnya.
12.Cetak.php ini digunakan untuk mencetak struk pada saat berbelanja 
13.Logout.php digunakan untuk mengakses menu utama yaitu login
